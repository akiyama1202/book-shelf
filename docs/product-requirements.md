# プロダクト要求定義書

## 1. プロダクトビジョンと目的

読んだ本・読みたい本を個人で記録・管理するための、シンプルな書籍管理システムである。
複雑な書誌情報管理は行わず、最低限の情報で「自分の読書記録を一覧・検索・整理できる」ことを目的とする。

## 2. ターゲットユーザーと課題・ニーズ

### ターゲットユーザー
- 個人で読書記録をつけたいユーザー

### 課題・ニーズ
- 読みたい本・読んだ本をリスト化して管理したい
- 過去に読んだ本の感想や評価を後から振り返りたい
- タイトルや著者から目的の本を素早く見つけたい
- 自分専用のアカウントでデータを管理したい（他人のデータと混在しない）

## 3. 主要な機能一覧

- ユーザー登録・ログイン機能
- 書籍の登録・編集・削除・詳細表示（CRUD）
- 書籍一覧表示（表形式・ページネーション付き）
- タイトル・著者によるフリーテキスト検索
- 並び替え（タイトル・著者・登録日・評価）
- 読書ステータス管理（読みたい / 未読 / 読書中 / 読了）
- 評価（1〜5段階、任意）
- タグ付け（自由入力・複数可）
- タグ管理（登録・編集・削除）
- メモ・感想の記録

## 4. 成功の定義

- ユーザーが自分の本棚情報をログイン後に登録・編集・削除できる
- 登録した本を一覧で確認し、タイトル・著者で検索できる
- 任意の項目で並び替えができる
- 各ユーザーのデータが他ユーザーから分離されている

## 5. ビジネス要件

- 個人利用を主目的としたツールであり、商用要件は特に定めない
- 外部の書誌情報API（ISBN検索等）との連携は行わない（MVPスコープ外）

## 6. ユーザーストーリー

- ユーザーとして、アカウントを作成してログインしたい。これにより自分専用の本棚を管理できる。
- ユーザーとして、本を登録したい。これにより読みたい本・読んだ本を記録できる。
- ユーザーとして、登録した本の情報を編集したい。これにより読書ステータスや評価・感想を更新できる。
- ユーザーとして、不要になった本の記録を削除したい。これにより本棚を整理できる。
- ユーザーとして、本の一覧をページ送りで確認したい。これにより蔵書数が多くても見やすく管理できる。
- ユーザーとして、タイトルや著者で本を検索したい。これにより目的の本を素早く見つけられる。
- ユーザーとして、タイトル・著者・登録日・評価で並び替えたい。これにより様々な視点で本棚を眺められる。
- ユーザーとして、本にステータス（読みたい/未読/読書中/読了）を設定したい。これにより読書進捗を管理できる。
- ユーザーとして、本に評価（1〜5段階）をつけたい。これにより読了後の満足度を記録できる。評価は任意項目とする。
- ユーザーとして、本に複数のタグを付けたい。これにより自分なりのジャンル分けで整理できる。
- ユーザーとして、登録済みのタグの名前を変更・削除したい。これにより自分のタグ整理ができる。
- ユーザーとして、本にメモ・感想を残したい。これにより読書体験を記録できる。

## 7. 受け入れ条件

- ログインしていない状態では書籍データの操作・閲覧ができない
- 登録済みユーザーは自分が登録した書籍のみを閲覧・編集・削除できる
- 書籍登録時にタイトル・著者・ステータスが必須項目として検証される
- 評価未入力でも書籍を登録・更新できる
- 一覧画面はページネーションされ、表形式で全項目（タイトル・著者・ステータス・評価・タグ・メモ）が表示される
- 一覧画面でタイトル・著者によるフリーテキスト検索ができる
- 一覧画面でタイトル・著者・登録日・評価による並び替えができる
- タグの編集・削除は自分が登録したタグのみ可能で、タグを削除した場合は関連する書籍からもそのタグが外れる

## 8. 機能要件

### 8.1 認証・ユーザー管理
- ユーザー登録（サインアップ）
- ログイン・ログアウト
- 認証されたユーザーのみ書籍データにアクセス可能

### 8.2 書籍管理（CRUD）
- 書籍の新規登録
- 書籍情報の編集
- 書籍の削除
- 書籍の詳細表示

### 8.3 書籍データ項目
| 項目 | 必須/任意 | 説明 |
|---|---|---|
| タイトル | 必須 | 書籍名 |
| 著者 | 必須 | 著者名 |
| ステータス | 必須 | 読みたい / 未読 / 読書中 / 読了 |
| 評価 | 任意 | 1〜5段階。ステータスに関わらず入力可能（システムによる制約は設けない） |
| タグ | 任意 | 自由入力、複数登録可 |
| メモ・感想 | 任意 | 自由記述 |
| 登録日 | 自動設定 | 並び替えに使用 |

#### ステータスの定義
- **読みたい**: まだ所有していない・購入前の本
- **未読**: 所有しているが、まだ読み始めていない本
- **読書中**: 現在読み進めている本
- **読了**: 読み終わった本

### 8.4 タグ管理
- タグの一覧表示
- タグの新規登録
- タグ名の編集（リネーム）
- タグの削除（削除すると、紐づく書籍からもタグが外れる）

### 8.5 一覧・検索・並び替え
- 表形式の一覧表示（全項目表示）
- ページネーション
- タイトル・著者によるフリーテキスト検索
- タイトル・著者・登録日・評価による並び替え（昇順・降順）

## 9. 非機能要件

- 個人利用規模を想定した性能要件（大規模アクセスは想定しない）
- ログイン情報・パスワードは安全に保管する（ハッシュ化等）
- 自分以外のユーザーの書籍データへアクセスできないようにアクセス制御を行う
- 入力値に対するバリデーション（XSS・不正値対策含む）を行う
