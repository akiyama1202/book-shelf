# ユビキタス言語定義

本ドキュメントは、プロダクト内で使用するドメイン用語・ビジネス用語・UI/UX用語について、日本語・英語表記およびコード上の命名を統一するためのものである。
要件定義・設計・実装のすべてにおいて、ここで定義した用語を使用する。

## 1. ドメイン用語・ビジネス用語

| 日本語 | 英語（コード上の命名） | 定義 |
|---|---|---|
| ユーザー | User | システムにログインして書籍を管理する人。1ユーザーにつき1つの本棚を持つ |
| 書籍 / 本 | Book | ユーザーが登録する読書記録の対象 |
| タイトル | Title | 書籍の名称（必須項目） |
| 著者 | Author | 書籍の著者名（必須項目） |
| ステータス | Status | 書籍の読書状況を表す区分（必須項目） |
| 評価 | Rating | 書籍に対する満足度。1〜5の整数、未設定可（任意項目） |
| タグ | Tag | 書籍を分類するためのラベル。ユーザーごとにマスタ管理し、複数付与可能 |
| メモ / 感想 | Memo | 書籍に対する自由記述のメモ・感想（任意項目） |
| 登録日 | CreatedAt | 書籍をシステムに登録した日時。並び替えに使用 |
| 更新日 | UpdatedAt | 書籍情報を最後に更新した日時 |
| 本棚 | Bookshelf | ユーザーが管理する書籍の集合全体を指す概念上の用語（DB上のテーブルとしては存在しない） |

## 2. ステータスの値

| 日本語（表示名） | 英語（コード上の値） | 定義 |
|---|---|---|
| 読みたい | want_to_read | まだ所有していない・購入前の本 |
| 未読 | unread | 所有しているが、まだ読み始めていない本 |
| 読書中 | reading | 現在読み進めている本 |
| 読了 | finished | 読み終わった本 |

## 3. 機能・操作に関する用語

| 日本語 | 英語（コード上の命名） | 定義 |
|---|---|---|
| ユーザー登録 | Register / SignUp | 新規ユーザーがアカウントを作成すること |
| ログイン | Login | 登録済みユーザーが認証情報を用いてシステムにアクセスすること |
| ログアウト | Logout | ログイン状態を終了すること |
| 書籍登録 | CreateBook | 新しい書籍を本棚に追加すること |
| 書籍編集 | UpdateBook | 既存の書籍情報を変更すること |
| 書籍削除 | DeleteBook | 書籍を本棚から削除すること |
| 書籍一覧 | BookList | 登録済み書籍を表形式で表示する画面・データ |
| 書籍詳細 | BookDetail | 1冊の書籍の全項目を表示する画面 |
| 検索 | Search | タイトル・著者に対するフリーテキスト検索 |
| 並び替え | Sort | タイトル・著者・登録日・評価による一覧の順序変更 |
| ページネーション | Pagination | 一覧を複数ページに分割して表示する仕組み |
| タグ管理 | TagManage | タグの登録・編集（リネーム）・削除を行う機能 |

## 4. UI/UX用語

| 日本語 | 英語（コード上の命名） | 定義 |
|---|---|---|
| ログイン画面 | LoginPage | ログインフォームを表示する画面 |
| ユーザー登録画面 | RegisterPage | アカウント作成フォームを表示する画面 |
| 書籍一覧画面 | BookListPage | 書籍一覧（表形式）を表示する画面 |
| 書籍登録画面 | BookCreatePage | 新規書籍の登録フォームを表示する画面 |
| 書籍編集画面 | BookEditPage | 既存書籍の編集フォームを表示する画面 |
| 書籍詳細画面 | BookDetailPage | 1冊の書籍の詳細を表示する画面 |
| タグ管理画面 | TagManagePage | タグの一覧・登録・編集・削除を行う画面 |

## 5. 命名規則との対応

詳細な命名規則（ファイル名・変数名・テーブル名・APIパス等の表記方法）は `docs/development-guidelines.md` の「2. 命名規則」を参照する。
本ドキュメントは「どの日本語にどの英語を当てるか」という対応関係を定義し、development-guidelines.md は「その英語をどう表記するか（snake_case/camelCase等）」を定義する。
